Expérience

Lors de cette expérience, l’objectif était de simuler une attaque par force brute. Une attaque par force brute consiste à tester toutes les combinaisons possibles pour deviner un mot de passe. Nous avons utilisé un script Python qui a testé automatiquement des combinaisons jusqu’à trouver le mot de passe "secret123". Cette simulation m’a permis de mieux comprendre les enjeux de sécurité informatique.
