📄 Description

Ce projet est une page web de type CV interactif qui présente :
- Une introduction « À propos de moi »
- Une section « Expérience »
- Une liste de « compétences » dynamiques que l'on peut « ajouter » ou « supprimer »

Les compétences sont stockées dans une base de données MySQL (`portfolio`) et sont gérées automatiquement par le script PHP.

---

## ⚙️ Fonctionnalités

Création automatique de la base de données `portfolio`  
Création automatique de la table `skills`  
Insertion de compétences de base (`Sérieux`, `Motivé`, `À l'écoute`, `Déterminé`) si aucune compétence n’est présente  
Formulaire pour **ajouter** une nouvelle compétence  
Lien **ajouter** pour **supprimer** une compétence
